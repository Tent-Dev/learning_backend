วิธีลงครั้งแรก
1) ถ้าใช้ จำลองServer แนะนำให้ใช้ Ampps แล้วสร้าง Domain ชื่อ learning.local ส่วน directory Folder ชื่อ learning
2) สร้าง Database ใน learning.local/phpmyadmin (หรือตามที่ตั้งไว้) ชื่อ Learning_db, เซ็ต utf8mb4_unicode_ci
3) นำไฟล์ learning_db.sql ในโฟเดอร์ SQL_FILE ไป import database ตามข้อ2
3) ถ้าใช้ตัวอื่น หรือต้องการชื่อ Domain อื่น เมื่อลงแล้วให้ไปแก้ไฟล์ system/class_db.php
	2.1) แก้บรรทัดแรก $BASE_domain = "http://learning.local"; เป็นชื่อ domain หรือ localhost ที่ทำในข้อ1
	2.2) แก้การเชื่อมต่อ บรรทัด function Main_db() ตามต้องการ เช่น  ชื่อ Databaseที่ใช้ (ถ้าเปลี่ยน)
4) ไฟล์ video ที่เคยอยู่ใน src จะถูกลบออกเพื่อลดขนาดไฟล์ แต่ค่าเริ่มต้นในระบบหลังบ้าน ไฟล์จะอยู่ใน upload/video/intro เมื่อทำการอัปโหลดจากระบบ
5) เข้าระบบหลังบ้าน ผ่าน http://learning.local/admin/login.php
